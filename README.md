# LBP COULEURS - Site Web Professionnel

## 🎨 À Propos

Site web moderne et haut de gamme pour LBP COULEURS, entreprise de plâtrerie et peinture à La Voulte-sur-Rhône.

## ✨ Caractéristiques

### Design
- ✅ Design moderne et épuré
- ✅ Palette de couleurs premium : blanc, noir, gris anthracite, touches beige
- ✅ Aspect professionnel et artisanal
- ✅ Responsive (mobile, tablette, desktop)
- ✅ Animations légères et modernes

### Sections

1. **Accueil**
   - Vidéo en arrière-plan
   - Slogan : "Entreprise de plâtrerie et peinture"
   - Bouton CTA "Demander un devis"

2. **Services** (8 services)
   - Plâtrerie
   - Pose de Placo
   - Faux Plafonds
   - Isolation
   - Peinture Intérieure
   - Peinture Extérieure
   - Rénovation
   - Conseil

3. **Galerie Avant/Après**
   - Slider interactif avant/après
   - 4 projets de référence
   - Animations fluides

4. **À Propos**
   - Présentation de l'entreprise
   - Valeurs et engagements
   - Statistiques (500+ projets, 15+ ans d'expérience, 98% satisfaction)

5. **Avis Google**
   - Affichage des avis Google
   - 5 étoiles intégrées
   - Témoignages clients

6. **Contact**
   - Formulaire complet (nom, téléphone, email, message)
   - Validation côté client
   - Intégration Formspree (emails fonctionnels)
   - Google Maps intégrée
   - Informations de contact
   - Réseaux sociaux

7. **Footer**
   - Contacts (téléphone, email)
   - Zone d'intervention
   - Liens réseaux sociaux

```css
    --color-anthracite: #3a3a3a;
    /* ... */

```

## 🚀 Déploiement

### GitHub Pages (Gratuit)
1. Allez dans Settings → Pages
2. Sélectionnez `main` branch
3. Votre site sera accessible à : `https://lbpcouleurs.github.io/lbpcouleurs-site`

### Netlify (Alternative)
1. Allez sur netlify.com
2. Connectez votre repo GitHub
3. Déploiement automatique !
